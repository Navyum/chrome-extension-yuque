YuqueOut - 语雀导出助手 | 一键批量导出语雀知识库

一键批量导出语雀知识库文档，支持 Markdown、Word、PDF、Excel 等多种格式，自动下载图片到本地，保留目录层级结构。

== 功能亮点 ==

✅ 智能导出 — 自动识别文档类型，匹配最佳导出格式
✅ 多格式支持 — 文档支持 Markdown / Word / PDF / JPG，表格和数据表支持 Excel
✅ 图片本地化 — Markdown 导出时自动下载 CDN 图片到本地 assets 目录
✅ 知识库管理 — 个人知识库 + 协作知识库，下拉多选一键导出
✅ 零配置鉴权 — 自动读取浏览器 Cookie，无需手动配置 Token
✅ 流程控制 — 支持暂停、继续、重试，导出过程完全可控
✅ 中英双语 — 支持中文和 English 界面切换
✅ 保留结构 — 按知识库目录层级保存文件，还原原始组织方式

== 支持的文档类型 ==

- 文档 (Doc): Markdown / Word / PDF / JPG
- 表格 (Sheet): Excel (.xlsx)
- 数据表 (Table): Excel (.xlsx)
- 画板 (Board): 语雀私有格式，暂不支持

== 使用步骤 ==

1. 在浏览器中登录语雀 (yuque.com)
2. 点击扩展图标，选择要导出的知识库（支持多选）
3. 选择导出格式，点击「开始导出」
4. 等待导出完成，文件自动保存到本地下载目录

== 隐私声明 ==

本扩展纯本地运行，不收集、不上传任何用户数据。Cookie 仅用于访问语雀 API 获取您的文档内容，不做任何存储或转发。所有导出操作均在您的浏览器中完成。
