YuqueOut - 语雀导出助手

从语雀批量导出知识库文档到本地。支持 Markdown、Word、PDF、Excel、CSV、PNG、SVG 等格式，方便迁移到 Obsidian、飞书、Notion 等工具或本地备份。

== 独有功能（语雀官方不提供） ==

- 所见即所得 — 只要你能打开文档，就能导出。无需导出权限，不依赖官方导出功能
- 收藏批量导出 — 一键导出你收藏的文档和知识库，官方无此功能
- 表格多格式导出 — 本地解压表格数据，支持 Excel/CSV/Markdown/HTML 四种格式（官方仅 Excel）
- 画板导出为图片 — 白板/画板导出为 PNG/JPG/SVG，官方 API 完全不支持
- 加密文档导出 — 输入密码后批量导出加密内容
- 浮动气泡一键导出 — 文档页面右侧浮动气泡，点击即导出当前文档
- 不依赖页面 UI — 直接调用 API + 本地引擎，语雀改版不影响使用

== 支持的文档类型 ==

- 文档 (Doc): Markdown / Word / PDF / JPG
- 表格 (Sheet): Excel / CSV / Markdown / HTML
- 画板 (Board): PNG / JPG / SVG
- 数据表 (Table): Excel

== 核心能力 ==

- 智能导出 — 自动识别文档类型，匹配最佳格式
- 双引擎 — 本地转换引擎 + 官方 API，可切换
- 图片本地化 — Markdown 导出自动下载图片到本地
- 批量操作 — 个人 + 协作知识库 + 收藏，多选一键导出
- 目录保留 — 按原始知识库结构保存文件
- 零配置 — 自动读取登录态，无需 Token
- 流程可控 — 暂停、继续、失败重试
- 中英双语 — 支持中文 / English 界面

== 使用方式 ==

1. 登录语雀 → 2. 选择知识库 → 3. 获取文件列表 → 4. 开始导出

== 隐私 ==

纯本地运行，不收集任何数据，不连接第三方服务器。
