YuqueOut - 语雀导出助手

从语雀批量导出知识库文档到本地。支持 Markdown、Word、PDF、Excel 等格式，方便迁移到 Obsidian、飞书、Notion 等工具或本地备份。

== 功能 ==

✅ 智能导出 — 自动识别文档类型，匹配最佳格式
✅ 多格式 — 文档: Markdown/Word/PDF/JPG，表格/数据表: Excel
✅ 图片本地化 — Markdown 导出自动下载图片到本地
✅ 批量操作 — 个人 + 协作知识库，多选一键导出
✅ 零配置 — 自动读取登录态，无需 Token
✅ 流程可控 — 暂停、继续、失败重试
✅ 保留结构 — 按原始目录层级保存

== 使用 ==

1. 登录语雀 → 2. 选择知识库 → 3. 点击导出

== 隐私 ==

纯本地运行，不收集任何数据。
