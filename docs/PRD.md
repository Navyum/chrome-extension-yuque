# 语雀知识库导出助手 — Chrome 扩展 PRD

## 一、产品概述

### 1.1 产品名称
**YuqueOut** — 语雀知识库一键导出助手

### 1.2 一句话定位
零门槛、一键式的语雀知识库批量导出 Chrome 扩展，让每个用户都能轻松带走自己的数据。

### 1.3 目标用户
| 用户群 | 占比 | 特征 | 核心诉求 |
|--------|------|------|---------|
| 技术型个人用户 | ~60% | 开发者、博主，100-500 篇文档 | 迁移到 Obsidian/博客，要 Markdown + 图片 |
| 非技术个人用户 | ~20% | 写作者、知识管理者，50-200 篇 | "一键操作"，不会命令行 |
| 团队/企业用户 | ~20% | 知识库管理者，500-5000+ 篇 | 协作库导出、迁移到 Confluence/飞书 |

### 1.4 市场机会
- 语雀注册用户 **1200 万+**，GitHub 导出工具生态累计 **5000+ stars**
- Chrome 商店仅 **1 个**直接竞品（XNotes，~1000 安装量），市场严重供给不足
- 现有 CLI 工具门槛高（需 Node.js/Python/Token），60%+ 用户被挡在门外
- **协作知识库导出**是所有工具的功能空白

### 1.5 核心差异化
| 差异点 | 我们 | CLI 工具（yuque-dl 等） | XNotes |
|--------|------|------------------------|--------|
| 安装门槛 | Chrome 商店一键安装 | 需 Node.js/Python 环境 | Chrome 商店 |
| 配置门槛 | 零配置（自动读取登录态） | 需手动获取 Token/Cookie | 零配置 |
| 协作库导出 | **支持** | 不支持 | 未知 |
| 增量导出 | 支持 | 部分支持 | 不支持 |
| 多格式 | Markdown + PDF + HTML | 仅 Markdown | 仅 Markdown |
| 附件下载 | 支持 | 仅 yuque-dl 支持 | 不支持 |
| 可视化操作 | 完整 GUI | 命令行黑框 | 有 GUI |

## 二、功能需求

### 2.1 功能优先级

#### P0 — MVP 必须（第一版上线）

**F1: 知识库列表加载**
- 打开插件 Popup 后，自动检测用户登录态
- 展示用户名下所有知识库（个人 + 协作）
- 每个知识库显示：名称、文档数量、最后更新时间
- 支持搜索/过滤知识库

**F2: 批量导出为 Markdown**
- 勾选一个或多个知识库
- 点击"导出"按钮，开始批量导出
- 导出内容：
  - 文档正文转为标准 Markdown
  - 保留知识库原始目录层级（文件夹结构）
  - 每篇文档包含 Front Matter（标题、创建时间、更新时间、标签）
- 导出为 ZIP 压缩包下载

**F3: 图片本地化**
- 自动下载文档中所有图片到 `assets/` 子目录
- 替换 Markdown 中的图片链接为本地相对路径
- 处理语雀 CDN 防盗链（携带 Referer 或使用 Cookie）

**F4: 实时进度展示**
- 显示当前导出进度：已完成/总数
- 显示当前正在处理的文档名称
- 导出完成后显示统计：成功数、失败数、跳过数
- 失败文档可点击重试

**F5: 登录态检测**
- 自动检测 yuque.com 的登录状态
- 未登录时提示"请先在浏览器中登录语雀"
- 登录过期时自动提示刷新

#### P1 — 第二版

**F6: 协作知识库导出**
- 展示用户参与的所有协作知识库
- 导出时自动处理权限（利用浏览器已有的访问权限）
- 对于只读权限的库，仍可导出文档内容

**F7: 增量导出**
- 记录每次导出的文档 ID + 更新时间
- 下次导出时对比，仅导出新增/修改的文档
- 可选择"全量导出"或"增量导出"
- 使用 chrome.storage.local 持久化导出记录

**F8: 附件下载**
- 文档中嵌入的附件（PDF、Word、Excel 等）一并下载
- 附件存放在对应文档同级的 `attachments/` 目录
- Markdown 中附件链接替换为本地路径

**F9: 导出设置**
- 可选择导出格式：Markdown（默认）/ HTML / PDF
- 可选择是否下载图片（默认开启）
- 可选择是否下载附件（默认关闭）
- 可设置请求间隔（默认 500ms，可调整 200ms-2000ms）
- 可设置图片并发下载数（默认 3）

#### P2 — 第三版

**F10: 单篇快捷导出**
- 在语雀文档页面注入"导出"按钮
- 点击后直接下载当前文档为 Markdown + 图片包

**F11: 画板/表格导出**
- 画板（lakeboard）导出为 PNG 图片
- 表格导出为 XLSX/CSV
- 数据表导出为 CSV

**F12: 导出到目标平台**
- 一键导出并格式化为 Obsidian vault 结构
- 一键导出并格式化为 Notion 导入包
- 一键导出并格式化为 VitePress 项目

**F13: 定时备份**
- 设置自动备份周期（每天/每周/每月）
- 后台静默增量备份
- 备份完成桌面通知

---

### 2.2 非功能需求

**性能**
- 单篇文档处理时间 < 2 秒
- 1000 篇文档的完整导出 < 30 分钟
- 图片并发下载，不阻塞文档处理

**稳定性**
- 请求失败自动重试（最多 3 次，指数退避）
- 导出中途可暂停/恢复
- 浏览器关闭后重新打开可继续未完成的任务

**安全**
- 不收集、不上传任何用户数据
- 所有处理在本地完成
- 不存储用户密码或 Token
- 仅读取 yuque.com 域名下的 Cookie

**兼容性**
- Chrome 88+
- Edge 88+（Chromium 内核）
- 支持语雀个人版和企业版
