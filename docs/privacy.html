隐私政策

最后更新：2025 年 4 月

1. 概述

YuqueOut 语雀导出助手（以下简称"本扩展"）尊重并保护用户的隐私。本隐私政策说明了本扩展如何处理您的数据。

2. 数据收集

本扩展不收集、不存储、不传输任何用户数据到外部服务器。具体来说：

  • 不收集您的个人身份信息
  • 不收集您的文档内容
  • 不收集您的浏览历史
  • 不使用任何第三方分析或跟踪服务
  • 不包含任何广告 SDK

3. 数据处理

本扩展的所有数据处理均在您的本地浏览器中完成：

  • Cookie 读取：仅读取 yuque.com 域名下的 _yuque_session Cookie，用于验证登录状态。该信息仅在本地使用，不会传输到任何外部服务器。
  • API 请求：直接向 yuque.com 发送请求获取您的知识库和文档数据。所有请求均通过您浏览器已有的登录态完成。
  • 文件下载：导出的文件直接下载到您指定的本地目录，不经过任何中间服务器。
  • 本地存储：使用 Chrome Storage API 存储您的设置偏好和导出记录，所有数据仅保存在您的浏览器本地。

4. 权限说明

  • cookies：读取 yuque.com 的登录 Cookie，用于验证登录状态
  • storage：保存设置和导出记录到浏览器本地
  • downloads：将导出的文件保存到您的下载目录
  • declarativeNetRequest：修改图片下载请求的 Referer 头，绕过语雀 CDN 的防盗链保护
  • alarms：支持定时自动备份功能

5. 第三方服务

本扩展不使用任何第三方分析、广告或数据收集服务。唯一的外部通信是直接与 yuque.com 及其 CDN（cdn.nlark.com、cdn.yuque.com）的 API 交互。

6. 数据安全

由于所有数据处理在本地完成，您的数据安全取决于您本地设备的安全性。我们建议：

  • 保持浏览器和操作系统更新
  • 不在公共计算机上使用本扩展
  • 定期检查已安装的浏览器扩展
