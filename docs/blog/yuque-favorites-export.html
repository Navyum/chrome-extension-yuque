语雀收藏夹怎么批量导出？收藏文档和知识库备份完整指南

2026-07-13 8 分钟阅读 约 2300 字 Kevin Yang

语雀官方没有收藏夹批量导出入口。YuqueOut 将个人收藏和团队空间收藏识别为「虚拟知识库」，可一键导出收藏的单篇文档、收藏的整个知识库和空间收藏内容。导出时支持 Markdown、图片本地化、目录结构保留；如果收藏内容需要密码，YuqueOut 会提示验证，验证成功后继续导出；如果文档没有导出权限，则自动切换本地转换引擎，按「能打开就能导出」逻辑保存到本地。

语雀收藏夹经常被当成「以后再看」的知识入口：别人分享的技术文章、团队里的规范文档、产品案例、项目复盘、外部协作库入口，都会被随手收藏。问题是，收藏越多，越容易变成新的信息孤岛。

如果有一天你要迁移平台、离职交接、整理个人知识库，才会发现：收藏夹能收集内容，却没有官方批量导出入口。本文聚焦一个具体问题：怎样把语雀收藏的文档和知识库完整备份到本地？

Key Takeaways
  • 语雀官方没有收藏夹批量导出功能，收藏内容需要单独处理
  • YuqueOut 会把收藏夹作为虚拟知识库展示，勾选即可批量导出
  • 收藏的别人文档、协作库文档、团队空间收藏都可以按本地引擎导出
  • Markdown + 图片本地化是收藏备份最推荐的组合

为什么语雀收藏难以导出？

语雀收藏夹的核心问题是：它是阅读入口，不是知识库。收藏夹里可能混合了单篇文档、整个知识库、团队空间文档和加密内容，但官方导出功能主要围绕「当前文档」或「自己的知识库」设计。

这会带来三个限制：

  • 没有批量入口：收藏列表没有「全部导出」按钮。
  • 来源复杂：收藏内容可能来自个人空间、协作库或团队空间，权限规则不同。
  • 格式不统一：有些收藏是文档，有些是知识库，有些还包含图片、表格或加密内容。

如果你手动处理，流程通常是打开收藏列表、逐个点开、判断类型、尝试导出、整理文件名，再处理图片裂图。收藏 20 篇还能勉强接受，收藏 200 篇就不现实了。

收藏夹里有哪些内容可以导出？

YuqueOut 支持把收藏夹中的两类内容纳入批量导出：

  • 收藏的单篇文档：按来源知识库或「单篇收藏」目录归类。
  • 收藏的整个知识库：获取知识库目录后，按原始层级导出全部可访问文档。

从空间范围看，它覆盖：

  • 个人收藏：你在个人空间中收藏的文档和知识库。
  • 团队空间收藏：团队空间独立收藏的文档和知识库。
  • 协作内容：你能访问但不一定拥有导出权限的协作库文档。

这也是 YuqueOut 和普通命令行导出工具的差异：收藏内容通常不在公开 API 的标准导出范围内，而 YuqueOut 使用浏览器登录态和本地转换引擎补齐了这部分缺口。

如何批量导出语雀收藏？

操作流程和普通知识库导出一致，不需要额外配置 Token：

  1. 安装 YuqueOut 扩展：从 Chrome 应用商店安装后，保持浏览器已登录语雀。

  2. 打开扩展弹窗：点击浏览器工具栏中的 YuqueOut 图标，等待知识库和收藏列表加载完成。

  3. 勾选「收藏」：个人收藏会显示在收藏分组下；团队空间收藏会显示在对应团队分组下。

  4. 获取文件信息：YuqueOut 会扫描收藏的单篇文档和收藏知识库中的文档目录。

  5. 选择格式并开始导出：推荐选择 Markdown，并开启图片本地化，方便长期保存和全文搜索。

如果你还没有做过完整知识库导出，可以先读 语雀知识库批量导出完整教程。收藏导出的交互和它基本一致，只是选择项从「知识库」变成了「收藏」。

导出后的目录结构

收藏导出最容易混乱的是目录。YuqueOut 会尽量保留来源信息，避免所有文档堆在同一个文件夹里。

语雀备份/
├── 个人空间/
│   └── 收藏/
│       ├── 单篇收藏/
│       │   ├── 前端性能优化清单.md
│       │   └── 项目复盘模板.md
│       └── 收藏的知识库名称/
│           ├── 01 入门.md
│           └── assets/
└── 团队名称/
    └── 收藏/
        ├── 产品规范库/
        └── 单篇收藏/

这套结构有两个好处：第一，你能看出内容来自个人收藏还是团队空间收藏；第二，收藏的整个知识库仍然保留原始目录层级，不会丢失上下文。

无导出权限和加密内容怎么处理？

收藏内容不等于拥有导出权限。很多收藏文档来自别人分享的协作库，你能阅读，但不一定能调用官方导出 API。

YuqueOut 的处理方式是：

  1. 先识别文档类型和可用导出路径。
  2. 如果官方导出受限，自动切换到本地 Lake 转 Markdown 引擎。
  3. 如果文档含图片，按设置下载图片并替换为本地相对路径。
  4. 如果收藏项需要密码，弹出密码验证窗口；验证成功后继续导出，用户也可以选择跳过。
安全说明

YuqueOut 不绕过加密保护。遇到加密文档或加密知识库时，必须由你输入正确密码完成验证。密码只用于向语雀完成验证，不会被上传到第三方服务器，也不会作为明文保存。

收藏导出推荐什么格式？

如果目标是长期保存和迁移，推荐组合是：Markdown + 图片本地化 + 本地转换引擎。

格式 适合场景 建议
Markdown 迁移到 Obsidian、Git、博客、知识库系统 首选
PDF 归档、审阅、固定版式分享 适合少量关键文档
Word 二次编辑、交付给非技术同事 按需选择

如果你遇到导出后图片打不开的问题，重点检查是否开启图片本地化。语雀 CDN 图片存在防盗链限制，具体原因可以参考 语雀图片防盗链解决方案。

常见问题

Q: 收藏夹里的外部链接也会导出吗？

YuqueOut 主要处理语雀内部文档和知识库。外部链接不是语雀文档，通常会作为链接信息保留，不会抓取第三方网页内容。

Q: 收藏的知识库会导出全部文档还是只导出入口？

如果你收藏的是整个知识库，YuqueOut 会获取该知识库的目录和文档列表，按可访问范围导出其中的文档，而不是只保存一个入口链接。

Q: 团队空间收藏和个人收藏会混在一起吗？

不会。团队空间收藏会按团队空间归类，个人收藏会放在个人空间下，导出目录能看出来源。
