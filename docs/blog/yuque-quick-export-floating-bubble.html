语雀单篇文档一键导出：浮动气泡功能怎么用？

2026-07-13 6 分钟阅读 约 1800 字 Kevin Yang

YuqueOut 的浮动气泡是语雀单篇文档快捷导出入口：安装扩展后，在语雀文档页面右侧会显示一个可拖拽圆形按钮，点击即可导出当前正在浏览的文档。导出格式、下载子目录、图片本地化和转换引擎都遵循设置页配置；如果当前文档没有官方导出权限，YuqueOut 会自动使用本地转换引擎导出为 Markdown。该功能适合临时保存单篇文章、补档收藏内容和快速备份协作文档。

批量导出适合迁移整个知识库，但日常使用里还有另一种高频场景：你正在浏览一篇语雀文档，觉得以后可能用得上，只想马上保存这一篇。

官方流程通常需要打开菜单、选择导出、等待生成、下载文件。YuqueOut 的浮动气泡把这个动作缩短成一次点击。

Key Takeaways
  • 浮动气泡用于导出当前正在浏览的单篇语雀文档
  • 导出格式和下载目录完全遵循 YuqueOut 设置页配置
  • 没有官方导出权限时，会自动走本地 Markdown 转换引擎
  • 不想显示气泡时，可在设置页关闭「显示浮动气泡」

什么时候需要单篇快捷导出？

以下场景特别适合使用浮动气泡：

  • 临时保存资料：看到一篇有价值的语雀文章，先保存到本地。
  • 补档收藏内容：批量导出后，后续又收藏了几篇，只想补导单篇。
  • 保存协作文档：你能阅读文档，但官方导出按钮不可用或权限不足。
  • 快速转 Markdown：把当前文档放进 Obsidian、VS Code 或 Git 仓库。
  • 检查导出效果：正式批量导出前，先用一篇文档测试格式和图片本地化。

如果你要迁移几百篇文档，仍然应该使用 批量导出流程。浮动气泡更像是「看到就存」的快捷入口。

浮动气泡怎么工作？

YuqueOut 会在语雀文档页注入一个隔离样式的浮动按钮。按钮只在文档页显示，不会出现在语雀首页、设置页或搜索页。

点击气泡后，扩展会识别当前页面的文档信息，并交给后台导出流程处理：

  1. 读取当前文档的 slug、知识库信息和页面上下文。
  2. 根据设置判断导出格式和转换引擎。
  3. 如果官方导出可用，则调用官方导出流程。
  4. 如果没有导出权限，则使用本地 Lake 转 Markdown 引擎。
  5. 按设置中的下载子文件夹保存到浏览器下载目录。

导出过程中，气泡会显示加载状态；成功后显示完成状态，失败时显示错误状态。这个反馈足够轻量，不会打断你继续阅读。

使用步骤

  1. 安装 YuqueOut：从 Chrome 应用商店安装扩展。

  2. 登录语雀：确保浏览器中已经登录 yuque.com。

  3. 打开任意语雀文档：进入文档正文页面，右侧会出现圆形浮动气泡。

  4. 点击气泡：不要拖动，直接点击即可开始导出当前文档。

  5. 查看下载结果：文件会保存到浏览器下载目录下的 YuqueOut 子文件夹中。

气泡可以拖动到页面左侧或右侧，松开后会自动吸附到最近边缘，避免遮挡正文。

相关设置

浮动气泡本身只有开关，但它会读取导出设置：

设置项 对浮动气泡的影响
文档默认导出格式 决定当前文档导出为 Markdown、Word、PDF 或 JPG
Markdown 转换引擎 决定 Markdown 走本地转换还是官方 API
图片本地化 决定 Markdown 中图片是否下载到本地 assets 目录
下载子文件夹 决定保存路径前缀

如果你不想显示气泡，可以打开 YuqueOut 设置页，在「快捷导出」区域关闭「显示浮动气泡」。刷新语雀文档页后设置生效。

没有导出权限也能用吗？

可以，只要你能打开并阅读文档。这是浮动气泡最有用的地方之一。

很多协作文档、别人分享的文档和收藏文章没有官方导出权限。官方按钮可能不可用，但页面内容已经在浏览器中渲染。YuqueOut 可以使用本地转换引擎读取可见内容，转成 Markdown 并保存到本地。

这和 「所见即所得」导出逻辑一致：能打开，才能导出；不能访问的内容不会被读取。

和批量导出的区别

能力 浮动气泡 批量导出
适合对象 当前单篇文档 整个知识库或收藏夹
操作成本 一次点击 先选择知识库，再扫描文件
典型用途 临时保存、补导单篇 迁移、备份、团队交接

简单说：少量、临时、当前页，用浮动气泡；大量、结构化、要完整目录，用批量导出。

常见问题

Q: 浮动气泡会不会遮挡文档内容？

气泡默认在页面右侧中部，并且可以拖动。拖动后松手会吸附到最近边缘，通常不会遮挡正文。

Q: 点击气泡后没有反应怎么办？

先确认当前页面确实是语雀文档页，并且已经登录语雀。如果仍然失败，可以打开扩展弹窗尝试批量导出，或查看 导出失败排查指南。

Q: 气泡导出会保存图片吗？

如果你选择 Markdown 且开启图片本地化，YuqueOut 会下载文档中的语雀 CDN 图片，并替换为本地相对路径。
