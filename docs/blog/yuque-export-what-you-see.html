语雀导出「所见即所得」：只要能打开就能导出

2025-04-25 8 分钟阅读 约 2200 字 Kevin Yang

YuqueOut 采用「所见即所得」导出逻辑：只要你能在浏览器中打开并阅读文档，就能将其导出到本地。不依赖语雀官方的导出权限设置——收藏的文档、协作知识库、他人分享的内容、甚至管理员禁止导出的文档都可以导出。原理是通过浏览器已有的登录态直接获取文档渲染内容，在本地完成格式转换，全程不调用语雀的导出 API，因此不受权限限制。

你的文档真的是你的吗？

我在语雀上写了三年文档。个人笔记、技术博客、读书摘要、项目文档......零零总总 400 多篇，散落在七八个知识库里。一直觉得挺好，云端同步，随时能看。

直到有一天，团队决定迁移到飞书。

领导一句"下周把语雀的文档都搬过来"，我才第一次认真研究语雀的导出功能。然后发现——

语雀居然没有"批量导出"按钮。

要导出，你得一篇一篇来：打开文档 → 点右上角更多 → 点导出 → 选格式 → 等待 → 下载。400 篇文档，每篇至少 30 秒点击等待，你算算要多久。

语雀的导出权限限制有多严格？

非常严格——没有导出权限的文档完全无法导出，收藏夹没有导出入口，画板不支持保存为图片。

试着导了几篇之后，我发现坑更多：

1. 别人分享的文档，根本导不出来

同事分享给我的协作库文档，我能打开、能阅读、能评论——但点"导出"，直接提示没有权限。

我明明能看到所有内容，凭什么不能存一份到本地？

2. 收藏夹里的文章，没有任何导出入口

我收藏了上百篇别人写的好文章，想趁周末整理一下。结果发现语雀的收藏功能只管"收"，不管"出"。没有批量导出，连单篇导出都不支持。

3. 表格只能导出 Excel，画板完全导不出

语雀的表格只能导出 xlsx 格式。如果我只是想要一个 CSV 或者 Markdown 表格呢？不好意思，没有。

画板（白板/思维导图）更惨——官方压根不支持导出为图片。你在画板上精心画的流程图，想保存一张 PNG？官方：不支持。

4. GitHub 上的 CLI 工具，劝退了我

搜了一圈开源工具，确实有几个能批量导出的——但要装 Node.js 环境、手动获取 API Token、在命令行里敲命令。对于不写代码的同事来说，这和没有一样。

什么是「所见即所得」导出？

「所见即所得」导出的核心逻辑是：只要你能在浏览器里打开并阅读一篇文档，就能将它导出为本地文件，完全不依赖语雀的导出权限设置。

后来我换了个思路：既然我能在浏览器里看到这篇文档的全部内容，为什么不能直接把它保存下来？

浏览器里看到的内容 = 可以保存的内容。

这就是 YuqueOut 的核心理念——所见即所得，只要你能打开，就能导出。

不需要导出权限，不需要管理员授权，不需要 API Token。你在语雀上能阅读的每一篇文档、每一张表格、每一个画板，YuqueOut 都能帮你保存到本地。

YuqueOut 是怎么绕过导出限制的？

YuqueOut 自动复用浏览器的登录状态读取文档内容，在本地完成格式转换，全程不调用语雀的导出 API，因此不受导出权限限制。

装好扩展之后，我的导出过程是这样的：

第一步：点开扩展，自动列出我所有的知识库（个人的、协作的、收藏的，全都有）。

第二步：全选，点"获取文件信息"。30 秒不到，342 篇文档的列表就出来了——包括收藏夹里那些别人写的文章。

第三步：点"开始导出"。然后泡杯咖啡，等着就行。

文档自动按知识库的目录结构存放，图片自动下载到本地。半小时后，我打开"下载"文件夹里的"语雀备份"目录——所有文档整整齐齐地躺在那里。

遇到几篇加密文档，扩展弹出了密码输入框，输入后继续导出。遇到没有导出权限的协作文档，扩展自动切换到本地转换，照样导出了 Markdown。

全程没有打开过一篇文档，没有点过一次"更多"菜单。

技术原理

传统导出工具依赖语雀的「导出 API」，受服务端权限控制。YuqueOut 走了一条不同的路：直接在浏览器中解析语雀页面返回的文档数据（Lake 格式），通过本地转换引擎将其转为 Markdown/Word/PDF。这意味着只要浏览器能渲染文档内容，YuqueOut 就能完成转换——完全绕过了服务端的导出权限校验。

YuqueOut 能导出哪些别人做不到的内容？

收藏夹批量导出、画板导为 PNG/JPG/SVG 图片、表格导为 CSV/Markdown——这些是目前只有 YuqueOut 能做到的。

用了一段时间后，我发现 YuqueOut 解决的不只是"批量导出"这一个问题。有几个场景让我特别惊喜：

浏览文档时随手保存

语雀文档页面右侧有一个小小的浮动气泡（可以拖动位置，也可以在设置里关掉）。看到一篇好文章？点一下气泡，直接保存为 Markdown 到本地。不需要打开扩展弹窗，不需要任何额外操作。

就像浏览器的"另存为"，但聪明 100 倍。

表格导出为 CSV / Markdown

语雀的表格只支持导出 Excel。但很多时候我只是想要一个 CSV 导入到其他工具，或者一个 Markdown 表格贴到文档里。YuqueOut 支持把表格导出为 Excel、CSV、Markdown、HTML 四种格式。

画板终于能保存图片了

这是我最满意的功能。语雀的画板（流程图、思维导图、白板）官方完全不支持导出为图片。YuqueOut 可以把画板导出为 PNG、JPG、SVG。

我的项目流程图、团队的脑暴思维导图，终于不用截屏了。

使用导出工具安全吗？数据会被上传吗？

完全安全。YuqueOut 所有操作 100% 在浏览器本地完成，不收集文档内容，不上传任何数据到第三方服务器。

可能有人会问：这个扩展会不会把我的文档上传到什么服务器？

答案是：不会。所有操作 100% 在你的浏览器本地完成。

  • 不收集任何文档内容
  • 不上传任何用户数据
  • 不需要 API Token 或密码（自动复用浏览器的登录状态）
  • 不连接任何第三方服务器

代码完全开源，托管在 GitHub 上，不放心的话可以自己看。
