YuqueOut - 语雀导出助手

语雀批量导出工具。登录语雀后，可导出你有权访问的知识库与收藏文档，无需手动配置 Token，一键备份到本地，支持多格式导出，方便迁移到 Obsidian、飞书、Notion。

核心优势

✅ 无需 Token 配置：复用浏览器中的语雀登录状态，上手零门槛
✅ 团队空间：支持你有权访问的团队空间批量导出
✅ 批量导出：支持知识库、收藏文档一键批量导出
✅ 浮动气泡：在语雀文档页面显示可拖拽快捷按钮，快速导出当前正在浏览的文档
✅ 多格式适配：默认 Markdown，表格支持 MD、CSV，画板导出增加 SVG
✅ 加密内容支持：需要密码的文档可输入密码后导出，协作库顺畅备份
✅ 结构完整：保留原有目录，图片自动本地化，减少链接失效
✅ 隐私安全：导出处理在本地浏览器完成，不上传、不收集文档内容

极简操作：
登录语雀 → 选择知识库/收藏 → 一键批量导出，无需额外 Token 配置。

适用场景：
语雀文档本地备份、收藏资料批量整理、跨平台知识迁移、团队文档归档留存。
