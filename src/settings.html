快捷导出

性能调优

仅控制单篇 Markdown 文档内图片本地化时的下载并发。当前实际最大并发为 3，建议 1-3。

关于 YuqueOut

零门槛、一键式的语雀知识库批量导出工具

项目说明

YuqueOut 是一款专为语雀用户打造的 Chrome 扩展，帮助你一键批量导出知识库内容。无论是个人笔记迁移到 Obsidian，还是团队知识库的定期备份，YuqueOut 都能轻松搞定。

所有数据处理完全在本地完成，不收集、不上传任何用户数据。

功能特性

知识库管理自动获取个人 + 协作知识库列表，支持多选批量导出
多格式导出Markdown（含 Front Matter）和 HTML 格式，保留完整目录层级
图片本地化自动下载语雀 CDN 图片，智能处理防盗链，替换为本地路径
附件下载文档中嵌入的 PDF、Word 等附件一并下载保存
增量导出记录导出历史，仅处理新增/修改的文档，节省大量时间
定时备份设置每天/每周/每月自动增量备份，数据安全无忧
协作库支持导出你参与的所有协作知识库和团队空间文档
隐私安全纯本地处理，零数据上传，不存储任何密码或 Token
